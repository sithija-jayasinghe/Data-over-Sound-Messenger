# Roadmap

## Short-Term Goals
- Complete core functionality (encoding, decoding, checksum).
- Develop a user-friendly interface.
- Write unit and integration tests.

## Medium-Term Goals
- Optimize FFT implementation for performance.
- Improve noise filtering algorithms.
- Add support for multi-language text.

## Long-Term Goals
- Extend frequency range for higher data rates.
- Explore real-time applications.
- Integrate with IoT devices.
